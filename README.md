# Java Portfolio Projects

Colección de proyectos desarrollados en Java enfocados en lógica de programación, desarrollo backend, consumo de APIs y gestión de datos.

Este repositorio reúne diferentes ejercicios y aplicaciones desarrolladas como parte del fortalecimiento de habilidades en programación orientada a objetos, estructuras de datos y arquitectura backend.

---

## 🚀 Tecnologías utilizadas

- Java
- Spring Boot
- Maven
- PostgreSQL
- JPA / Hibernate
- REST APIs
- JSON
- Git & GitHub

---

## 📂 Contenido del repositorio

| Proyecto | Descripción |
|---|---|
| Consumo API | Integración y consumo de APIs externas |
| Gestión de Libros | Aplicación backend para administración de libros |
| Persistencia de Datos | Uso de PostgreSQL con JPA |
| Menú por Consola | Aplicaciones interactivas CLI |
| Programación Orientada a Objetos | Implementación de clases y objetos |

---

## 🧩 Funcionalidades implementadas

- Consumo de APIs REST
- Conversión y serialización JSON
- Persistencia en base de datos
- Arquitectura por capas
- Manejo de entidades
- Consultas SQL
- Validaciones básicas
- CRUD completo

---
